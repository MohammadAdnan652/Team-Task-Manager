# Team Task Manager

A full-stack application for managing team projects and tasks.

## Quick Start

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Access the app at `http://localhost:3000`

## Features

- User Authentication (JWT)
- Project Management
- Task Assignment & Tracking
- Dashboard with Statistics
- Role-based Access Control

## Tech Stack

- Backend: Node.js, Express, MongoDB
- Frontend: React, Vite, Tailwind CSS
- Auth: JWT, Bcrypt
